Walkthrough

Check the front drawer. The paper tells you the order
of the numbers for the code.

First Number Answer: Check the right table and solve for yellow.
√144 - 3 = 9. So the first number is 9.

Second Number Answer: Go back to the front Drawer. You can
open the drawer. Inside, you see two sheets of paper. Follow the right
paper's instructions. If you trace it out, you will get the number 5.

Third Number Answer: Go to the shelf. You will see that there
is an empty spot on the shelf. It is the second row, second column.
Therefore, the answer is 2.

Now go back to the front drawer. Go to the password box
and insert the code 9-5-2.

Go back to view the Entire Room
