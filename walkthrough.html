Walkthrough

Check the Front Drawer. Then Check the Paper. The paper tells you the order of the numbers for the password code.

1st Num: "Color Pencil" in yellow -- Check the Right Table. Find the corresponding color.
2nd Num: "Paper in the drawer below" -- Check the Front Drawer. Open the drawer.
3rd Num: "Check the shelf" -- Check the Left Shelf and find the hint.

First Number Answer:
Check the right table and solve for yellow. √ 144  - 3 = 9.
So the first number is 9.

Second Number Answer:
Go to the "Front Drawer". You can open the drawer. Inside,
you see two sheets of paper. Follow the right paper's instructions.

If you trace out the order of the numbers/letters on the left paper,
you will get the number 5.

Third Number Answer:
Go to the shelf. You will see that there is an empty spot on the shelf.
It is the second row, second column, coodordinate (2,2).

Therefore, the answer is 2.

Unlock the Password Box
Now go back to the front drawer. Go to the Password Box.
Click on the numbers on the screen in the order of 9-5-2.

Bonus Color Box Solution
Go the Entire Room View. Check the Light on the Wall.

Go back to view the Entire Room
